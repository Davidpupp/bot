# Banners do Bot

## Como usar:

1. Coloque sua imagem de banner aqui com o nome exato: `welcome_banner.jpg`
2. Formatos aceitos: `.jpg`, `.jpeg`, `.png`
3. Tamanho recomendado: 800x400 pixels (ou proporcional)
4. Tamanho máximo do arquivo: 10MB

## Exemplo:

- Se você tiver um arquivo chamado `minha_loja.png`, renomeie para `welcome_banner.jpg`
- Ou copie e cole sua imagem com o nome `welcome_banner.jpg`

## Funcionamento:

- Se o banner existir, ele será enviado junto com a mensagem de boas-vindas
- Se não existir, apenas a mensagem de texto será enviada
- O sistema tem fallback automático em caso de erro no envio da imagem
